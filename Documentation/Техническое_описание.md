# Техническое описание проекта «Выборка из Мониторинга»

## Цель

Разработать автономный VBA-макрос для Excel, который формирует выборку закупочных процедур из файлов мониторинга УЗРКС за выбранные пользователем годы.

## Совместимость

- Excel 2016 / 2021 / 2024.
- 32-bit и 64-bit версии Excel.
- Внешние подключения не требуются.

## Расположение файлов

Основной файл с макросом:

```text
Рабочий стол\Выборка из Мониторинга\Выборка из Мониторинга.xlsm
```

Файлы мониторинга рекомендуется хранить в папке:

```text
Рабочий стол\Выборка из Мониторинга\Source\Мониторинг УЗРКС
```

Поддерживаемый шаблон имени:

```text
2021 Мониторинг УЗРКС*.xls*
2022 Мониторинг УЗРКС*.xls*
2023 Мониторинг УЗРКС*.xls*
2024 Мониторинг УЗРКС*.xls*
2025 Мониторинг УЗРКС*.xls*
2026 Мониторинг УЗРКС*.xls*
```

## Листы

| Лист | Назначение |
| --- | --- |
| `Закупщик` | источник данных в файлах мониторинга |
| `Отчет УЗРКС` | результат выборки в файле с макросом |

## Варианты поиска

1. По номеру закупки: колонки 3, 4, 12, 32 и дополнительные SAP-колонки по годам.
2. По наименованию закупки: колонка 11, частичное совпадение.
3. По номеру ЦЗК: колонка 58, поиск без учета регистра и разделителей.
4. По типу сделки: колонка 58, поиск без учета регистра и разделителей.

## Журнал ошибок

Журналы создаются в папке `Logs` в формате:

```text
Ошибки_YYYYMMDD_HHMMSS.txt
```

Фиксируются отсутствующие файлы, ошибки открытия, отсутствие листов и пустые источники.
