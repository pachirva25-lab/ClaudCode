# VBA-модуль «Выборка из Мониторинга»

Файл `SelectionMonitoring.bas` содержит макрос `СформироватьВыборкуУЗРКС` для Excel 2016 / 2021 / 2024 в 32-bit и 64-bit версиях.

## Что делает макрос

- Создает структуру папок проекта рядом с книгой макроса.
- Предлагает выбрать годы мониторинга: 2021–2026.
- Проверяет наличие файлов вида `202х Мониторинг УЗРКС*.xls*`.
- Предлагает выбрать один или несколько вариантов поиска:
  1. по номеру закупки;
  2. по наименованию закупки;
  3. по номеру ЦЗК;
  4. по типу сделки.
- Открывает исходные файлы только для чтения.
- Берет данные с листа `Закупщик`.
- Переносит найденные строки в лист `Отчет УЗРКС` текущей книги.
- Добавляет крайнюю колонку `Год мониторинга`.
- Создает журнал ошибок в папке `Logs`.

## Если файла нет локально

Если в папке `ClaudCode` на вашем компьютере нет папки `vba`, а видны только `.git`, `source-files` и `.gitkeep`, значит последние изменения еще не загружены из GitHub. Получите их через `Fetch origin` / `Pull origin` в GitHub Desktop или скачайте файл вручную. Подробная инструкция находится в `GET_SELECTION_MONITORING_BAS.md`.

Правильное расширение файла — `.bas`, не `.buzz`.

## Где находится файл модуля

Файл, который нужно импортировать в Excel, находится в этом проекте по пути:

```text
vba\SelectionMonitoring.bas
```

Если в окне `Import File` вы видите только файлы `VBA_Macro_Generator` и `VBA_Macro_Generator_FULL_FIXED`, значит открыта другая папка, например `Рабочий стол\Cursor_VBA`. Нужно перейти в папку данного репозитория `ClaudCode`, затем открыть подпапку `vba`.

В Windows файл может отображаться как `SelectionMonitoring` без расширения `.bas` — это нормально, если в колонке `Тип` указано `Файл "BAS"`.

## Как установить модуль в Excel

1. Откройте файл `Выборка из Мониторинга.xlsm`.
2. Нажмите `Alt + F11`, чтобы открыть редактор VBA.
3. В меню выберите `File → Import File...`.
4. В окне импорта перейдите в папку проекта:

```text
...\ClaudCode\vba
```

5. Выберите файл `SelectionMonitoring.bas` или `SelectionMonitoring` с типом `Файл "BAS"`.
6. Нажмите `Open / Открыть`.
7. Сохраните книгу в формате `.xlsm`.

## Как подготовить файлы

Рекомендуемая папка проекта:

```text
Рабочий стол\Выборка из Мониторинга
```

Файлы мониторинга положите в:

```text
Рабочий стол\Выборка из Мониторинга\Source\Мониторинг УЗРКС
```

Имена файлов должны начинаться так:

```text
2021 Мониторинг УЗРКС
2022 Мониторинг УЗРКС
2023 Мониторинг УЗРКС
2024 Мониторинг УЗРКС
2025 Мониторинг УЗРКС
2026 Мониторинг УЗРКС
```

Расширение может быть `.xlsx`, `.xlsm` или `.xls`.

## Как запустить

1. Вернитесь в Excel.
2. Нажмите `Alt + F8`.
3. Выберите макрос `СформироватьВыборкуУЗРКС`.
4. Нажмите `Run / Выполнить`.
5. Последовательно ответьте на вопросы в диалоговых окнах.

## Как проверить результат

1. Откройте лист `Отчет УЗРКС`.
2. Убедитесь, что появились найденные строки.
3. Проверьте крайнюю колонку `Год мониторинга`.
4. Если строк нет или были ошибки, откройте последний файл в папке `Logs`.
