# Выборка из Мониторинга

Новый проект VBA для Excel по формированию выборки закупочных процедур из файлов мониторинга УЗРКС.

## Основание

Проект скорректирован по загруженному тексту промпта и ТЗ:

- целевой файл: `Выборка из Мониторинга.xlsm`;
- источник: файлы `2021 Мониторинг УЗРКС` — `2026 Мониторинг УЗРКС`;
- лист-источник: `Закупщик`;
- лист результата: `Отчет УЗРКС`;
- платформа: Excel 2016 / 2021 / 2024, 32-bit и 64-bit;
- обработка полностью локальная, без внешних подключений.

## Что реализовано

- `vba/SelectionMonitoring.bas` — VBA-модуль с макросом `СформироватьВыборкуУЗРКС`.
- Выбор годов мониторинга 2021–2026.
- Проверка отсутствующих файлов с записью в журнал `Logs`.
- Выбор одного или нескольких вариантов поиска.
- Поиск по номеру закупки, наименованию закупки, номеру ЦЗК и типу сделки.
- Перенос найденных строк на лист `Отчет УЗРКС`.
- Добавление колонки `Год мониторинга`.
- Документация пользователя и техническое описание.
- Общие VBA-подходы в папке `Skills`.
- Папка `Домашнее задание` с перечнем файлов и скриншотов для сдачи.

## Если вы не видите `SelectionMonitoring.bas`

Если в вашей локальной папке `ClaudCode` видны только `.git`, `source-files` и `.gitkeep`, значит последние изменения с VBA-модулем еще не подтянуты на компьютер. Откройте `GET_SELECTION_MONITORING_BAS.md` и выполните шаги по получению файла.

Если на GitHub во вкладке `Pull requests` пусто, значит PR с файлами еще не создан/не попал в ваш репозиторий. Вернитесь в Codex-задачу и нажмите `Создать PR` / `Create PR`, затем merge и pull.

Под «Codex-задачей» имеется в виду вкладка браузера с адресом вида `chatgpt.com/codex/cloud/tasks/...`, а не Проводник Windows, GitHub Desktop, VS Code или обычная страница GitHub.

## Быстрый старт для проверки макроса

1. Создайте на рабочем столе папку `Выборка из Мониторинга`.
2. Положите файл `Выборка из Мониторинга.xlsm` в эту папку.
3. Возьмите файл `SelectionMonitoring.bas` из папки репозитория `vba` и импортируйте его через `Alt + F11 → File → Import File...`.
4. Положите файлы мониторинга в `Source\Мониторинг УЗРКС`.
5. Запустите в Excel макрос `СформироватьВыборкуУЗРКС` через `Alt + F8`.
6. Проверьте лист `Отчет УЗРКС` и журнал в папке `Logs`.
